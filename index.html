Vaccination Bot
   

I developed this bot look for vaccine availability in nearby locations and alert a person on telegram app as soon as vaccine was available in his/her location.

Bot was developed to help frontline workers as vaccination was compulsary for their regular operation. Through this bot I have helped many people to get vaccinated at the time when vaccine availability was very less.

The project was built using python and telegram api.

Movie ticket booking system
   

Built digital solution for movie ticket booking system catering the ticket booking services to consumers and theatre management capabilities to theatre owners.

The project enabled users to book movie tickets, owners to manage theatres and realtime payments and refunds were also made possible with the help of Instamojo payment API's.

The project was built using php, mysql, html, css and bootstrap.

Courier management system
   

The project was built to support small scale courier business vendors to have an digital solution to manage their business and gain vital insights about business operations.

This digital solution enabled end to end courier services through internet without any paperwork or requiring to visit nearby courier service for information and also provided features like live tracking of courier.

The project was built using php, mysql, html, css , ionic and bootstrap.
